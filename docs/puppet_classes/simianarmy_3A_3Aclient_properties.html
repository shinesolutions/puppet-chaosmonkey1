Puppet Class: simianarmy::client_properties

Defined in:
manifests/client_properties.pp

Summary

Write the `client.properties` file.

Overview

This class writes the client.properties file for the Netflix Chaos Monkey. It is auto-generated and should not be modified by hand. Documentation for each of the properties can be found in the example client.properties from which it is generated. (see below)

Note: This is an internal class that shouldn't be used directly. It is included by the simianarmy class. You can change values for it's parameters using Hiera.

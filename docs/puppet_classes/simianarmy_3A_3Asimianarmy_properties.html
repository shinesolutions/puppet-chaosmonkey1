Defined in:

Summary

Write the `simianarmy.properties` file.

Overview

This class writes the simianarmy.properties file for the Netflix Chaos Monkey. It is auto-generated and should not be modified by hand. Documentation for each of the properties can be found in the example simianarmy.properties from which it is generated. (see below)

Note: This is an internal class that shouldn't be used directly. It is included by the simianarmy class. You can change values for it's parameters using Hiera.

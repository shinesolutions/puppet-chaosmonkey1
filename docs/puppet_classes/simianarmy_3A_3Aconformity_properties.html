Puppet Class: simianarmy::conformity_properties

Defined in:
manifests/conformity_properties.pp

Summary

Write the `conformity.properties` file.

Overview

This class writes the conformity.properties file for the Netflix Chaos Monkey. It is auto-generated and should not be modified by hand. Documentation for each of the properties can be found in the example conformity.properties from which it is generated. (see below)

Note: This is an internal class that shouldn't be used directly. It is included by the simianarmy class. You can change values for it's parameters using Hiera.
